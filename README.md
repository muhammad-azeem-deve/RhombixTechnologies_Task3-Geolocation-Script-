
# 🌍 GeoLocator Pro - IP Geolocation Tracker

GeoLocator Pro is a Python-based desktop application that allows users to input any IP address and retrieve its geographical location. The app displays detailed information and visualizes the location on an interactive map.

---

## 🚀 Features

- 🔎 Track any IP address
- 📍 Displays City, Country, Region
- 🌐 Shows ISP (Internet Service Provider)
- 🗺️ Interactive map using Folium
- 💻 Modern GUI built with Tkinter
- ⚡ Fast and lightweight

---

## 🛠️ Technologies Used

- Python
- Tkinter (GUI)
- Requests (API handling)
- Folium (Map visualization)

---

## 📦 Installation & Setup

1. Clone the repository:

git clone https://github.com/your-username/geolocator-pro.git
cd geolocator-pro

2. Install dependencies:
pip install requests folium

3. Run the application:
python app.py
